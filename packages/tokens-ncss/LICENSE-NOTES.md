# Font licensing — Avenir Next World

Working assumption: **NCSS holds a current Monotype/Linotype web font license that covers self-hosting Avenir Next World on NCSS properties.** This package ships the WOFF2 derivatives under that assumption.

The TTF originals were provided by the NCSS team on 2026-05-29 (see `brand-objects/` at the repo root).

## When to revisit this assumption

Re-check the license terms before:

- Adding a new domain that wasn't covered when the license was procured.
- Hitting pageview thresholds that escalate the license tier.
- Subsetting (some Monotype licenses forbid modification of the font files).
- Renewing — confirm the license is still current at renewal time.
